Help

Account issues

You User Manager can:

  • check and change the email address registered to your account
  • view and change the permissions on your account
  • set up new user accounts
  • delete user accounts

Changing your password

You can change your password by following the reset password process.

Skipping getting a security code

You can choose to sign in without a security code for up to 24 hours. Tick the box on the security code page when you sign in.

You should only skip getting a security code if you are working on a trusted device.

Automatically signing out

Bichard7 will sign you out if there is 30 minutes of no activity. You need to sign in again to access Bichard 7.

Something else is not working

Contact the Bichard7 support desk to report something not working correctly.
